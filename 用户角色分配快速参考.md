# 用户角色分配 - 快速参考指南

## [object Object]分钟快速上手

### 1️⃣ 文件位置

```
src/
├── api/user-role.js                    # API 接口
├── components/UserRoleDialog.vue       # 对话框组件
└── views/user/list.vue                 # 用户列表（已集成）
```

### 2️⃣ 核心 API

| 函数 | 用途 | 调用方式 |
|------|------|---------|
| `getUserRoles(userId)` | 获取用户角色 | `await getUserRoles(1)` |
| `assignRoles({userId, roleIds})` | 分配角色 | `await assignRoles({userId: 1, roleIds: [1,2]})` |
| `addUserRole(userId, roleId)` | 添加单个角色 | `await addUserRole(1, 2)` |
| `removeUserRole(userId, roleId)` | 删除单个角色 | `await removeUserRole(1, 2)` |
| `removeAllUserRoles(userId)` | 删除所有角色 | `await removeAllUserRoles(1)` |

## ⚠️ 注意事项

### 1. 权限检查

确保用户有以下权限：
- `admin:user-role:list` - 查看角色
- `admin:user-role:assign` - 分配角色

### 2. 后端配置

确保后端服务运行在正确的地址：
```javascript
// vite.config.js
proxy: {
  '/api': {
    target: 'http://localhost:8801'  // 修改为实际地址
  }
}
```

### 3. 数据验证

- 用户ID 必须存在
- 角色ID 必须有效
- 不能分配禁用的角色

|------|------|---------|
| 404 Not Found | API 路径错误 | 检查后端地址配置 |

| 500 Server Error | 后端异常 | 查看后端日志 |

## 📞 常见问题

**Q: 如何只添加角色而不删除原有角色？**  
A: 使用 `addUserRole()` 而不是 `assignRoles()`

**Q: 能分配多个角色吗？**  
A: 可以，使用多选即可

**Q: 分配失败怎么办？**  
A: 检查网络、权限、后端服务是否正常

**Q: 如何清除用户所有角色？**  
A: 在对话框中移除所有角色标签，点击确定即可
