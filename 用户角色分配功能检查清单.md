✅ 权限: admin:user-role:list
✅ 返回: UserRoleResponse[]
✅ 错误处理: 已实现
```

### 接口 2: 分配角色

```
✅ 接口名: assignRoles
✅ 方法: POST
✅ 路径: /user-role/assign
✅ 权限: admin:user-role:assign
✅ 请求体: { userId, roleIds }
✅ 返回: { code, message, data }
✅ 错误处理: 已实现
```

```
✅ 接口名: addUserRole

✅ 路径: /user-role/add/{userId}/{roleId}
✅ 权限: admin:user-role:add
✅ 返回: { code, message, data }
✅ 错误处理: 已实现
```

```
✅ 接口名: removeUserRole
