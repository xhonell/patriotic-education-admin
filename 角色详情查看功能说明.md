# 角色详情查看功能说明

## 📋 功能概述

实现了角色详情查看功能，通过 `/admin/role/detail/{id}` 接口获取角色完整信息，包括基本信息和已分配的权限树状结构，以美观的对话框形式展示。

## ✨ 功能特性

### 1. 角色基本信息展示
- 角色名称
- 角色状态（带颜色标签）
- 角色描述
- 创建时间
- 更新时间

### 2. 权限树状展示
- 树形结构显示已分配权限
- 区分菜单和按钮类型
- 显示权限标识码（code）
- 支持全部展开
- 空状态友好提示

### 3. 美观的视觉效果
- 渐变色对话框头部
- 卡片式信息布局
- 响应式网格布局
- 悬停交互效果

## 🔌 API 接口

### 获取角色详情
```http
GET /admin/role/detail/{id}
```

**路径参数**：
- `id`: 角色ID

**响应数据** (RoleDetailResponse)：
```javascript
{
  id: Long,                    // 主键
  name: String,                // 角色名称
  description: String,         // 角色描述
  status: Boolean,             // 状态（true 启用，false 禁用）
  createTime: LocalDateTime,   // 创建时间
  updateTime: LocalDateTime,   // 更新时间
  permissions: [               // 权限树列表
    {
      id: Long,                // 权限ID
      name: String,            // 权限名称
      code: String,            // 权限标识
      type: Byte,              // 权限类型（1 菜单 2 按钮）
      path: String,            // 路由路径
      parentId: Long,          // 父级ID
      status: Boolean,         // 状态
      sort: Integer,           // 排序
      childrenPermission: []   // 子权限（递归）
    }
  ]
}
```

## 💻 技术实现

### 前端 API 调用

**文件**: `src/api/role.js`

```javascript
// 获取角色详情（包含权限树）
export function getRoleDetail(id) {
  return request({
    url: `/admin/role/detail/${id}`,
    method: 'get'
  })
}
```

### 组件实现

**文件**: `src/views/content/role.vue`

#### 1. 数据定义

```javascript
// 查看详情对话框
const viewDialogVisible = ref(false)
const viewLoading = ref(false)
const viewData = reactive({
  id: null,
  name: '',
  description: '',
  status: true,
  createTime: '',
  updateTime: '',
  permissions: []
})

// 查看详情树形控件配置
const viewTreeProps = {
  label: 'name',
  children: 'childrenPermission'
}
```

#### 2. 查看功能

```javascript
// 查看
const handleView = async (row) => {
  viewDialogVisible.value = true
  viewLoading.value = true
  
  try {
    const res = await getRoleDetail(row.id)
    
    if (res) {
      Object.assign(viewData, {
        id: res.id,
        name: res.name,
        description: res.description,
        status: res.status,
        createTime: res.createTime,
        updateTime: res.updateTime,
        permissions: res.permissions || []
      })
      
      console.log('角色详情加载成功：', viewData)
    }
  } catch (error) {
    console.error('获取角色详情失败：', error)
    ElMessage.error('获取角色详情失败')
    viewDialogVisible.value = false
  } finally {
    viewLoading.value = false
  }
}
```

**功能特点**：
- ✅ 加载状态显示
- ✅ 错误处理
- ✅ 数据安全检查

## 🎯 使用场景

### 场景 1：查看角色完整信息
```
1. 在角色列表中点击"查看详情"按钮（眼睛图标）
2. 系统打开详情对话框，显示加载动画
3. 加载完成后显示：
   - 角色基本信息（名称、状态、描述、时间）
   - 权限树（已分配的所有权限）
4. 用户可以浏览权限树，查看具体权限
5. 点击"关闭"按钮关闭对话框
```

### 场景 2：查看空权限角色
```
1. 点击查看未分配任何权限的角色
2. 基本信息正常显示
3. 权限部分显示空状态提示
4. 图标 + "暂未分配任何权限"文字
```

### 场景 3：查看复杂权限结构
```
1. 点击查看拥有多级权限的角色
2. 权限树完全展开显示
3. 树形结构清晰展示父子关系
4. 可以看到每个权限的：
   - 名称
   - 类型（菜单/按钮）
   - 权限标识码
```

1. **权限树结构**
   - 使用 `childrenPermission` 字段
   - 支持多级嵌套
   - 自动过滤禁用权限

2. **空值处理**
   - description 为空显示 "-"
   - permissions 为空显示空状态
   - 时间为空显示 "-"

3. **加载状态**
   - 对话框打开立即显示 loading
   - 加载失败自动关闭对话框
   - 错误提示友好

4. **数据格式**
   - 时间格式：`yyyy-MM-ddTHH:mm:ss`
   - 需要替换 T 为空格显示
