# 年龄段管理功能说明 ⏱️

## 📋 功能概述

在内容管理模块中新增了**年龄段管理**功能，用于管理系统中的年龄段分类，为不同年龄层的用户提供适配的教育内容。

## 🎯 核心功能

### 1. **年龄段列表**
- ✅ 展示所有年龄段
- ✅ 显示年龄范围（最小年龄-最大年龄）
- ✅ 显示创建和更新时间
- ✅ 支持编辑和删除操作

### 2. **搜索筛选**
- 📝 按名称搜索

### 3. **年龄段管理**
- ➕ 新增年龄段
- ✏️ 编辑年龄段
- 🗑️ 删除年龄段

### 核心功能实现

#### 1. 年龄范围验证
```javascript
const validateAgeRange = (rule, value, callback) => {
  if (rule.field === 'maxAge' && value <= form.minAge) {
    callback(new Error('最大年龄必须大于最小年龄'))
  } else if (rule.field === 'minAge' && form.maxAge && value >= form.maxAge) {
    callback(new Error('最小年龄必须小于最大年龄'))
  } else {
    callback()
  }
}
```

#### 2. 数据加载
```javascript
const loadData = async () => {
  const params = {
    page: pagination.page,
    pageSize: pagination.pageSize,
    name: searchForm.name
  }
  const res = await getAgeRangeList(params)
  ageRangeList.value = res.list
  pagination.total = res.total
}
```

#### 3. 新增/编辑
```javascript
const handleSubmit = async () => {
  const submitData = {
    id: form.id,
    name: form.name,
    minAge: form.minAge,
    maxAge: form.maxAge,
    description: form.description
  }
  
  if (form.id) {
    await updateAgeRange(submitData)
  } else {
    delete submitData.id
    await addAgeRange(submitData)
  }
  loadData()
}
```

#### 4. 删除
```javascript
const handleDelete = (row) => {
  ElMessageBox.confirm('确定删除？')

      await deleteAgeRange(row.id)
      loadData()
    })
}
```

## 💡 使用建议

### 初始化数据
建议预先创建以下常用年龄段：

| 名称 | 年龄范围 | 说明 |
|------|---------|------|
| 幼儿 | 3-6岁 | 学前教育阶段，适合简单的爱国启蒙教育 |
| 儿童 | 6-12岁 | 小学阶段，适合基础爱国知识教育 |
| 少年 | 12-15岁 | 初中阶段，适合深入的历史和文化教育 |
| 青少年 | 15-18岁 | 高中阶段，适合系统的爱国主义教育 |
| 青年 | 18-35岁 | 大学及工作初期，适合实践性教育 |

### 年龄段划分原则
- 根据教育阶段划分（学前、小学、初中、高中等）
- 考虑认知发展水平
- 与教育内容难度相匹配
- 避免年龄段重叠

### 说明字段使用
- 建议填写该年龄段的教育特点
- 可以说明适合的内容类型
- 有助于内容编辑者选择合适的年龄段

## 🎯 应用场景

### 1. 内容推荐
- 根据用户年龄推荐合适的教育内容
- 确保内容难度与年龄相匹配

### 2. 内容分类
- 为教育内容标注适用年龄段
- 便于内容管理和筛选
