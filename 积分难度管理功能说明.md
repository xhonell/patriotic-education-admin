# 积分难度管理功能说明

## 📋 功能概述

积分难度管理模块用于管理系统中的积分难度等级，支持难度的增删改查操作。每个难度等级包含星级评定和对应的积分值，可用于题目分类和积分奖励设置。

## 🎯 功能特性

### 1. 难度列表展示
- ✨ 美观的表格布局，展示所有难度等级信息
- 🌟 星级可视化显示（1-5星）
- 🎨 状态标签（启用/禁用）
- 💯 积分显示
- 📝 创建和更新时间记录
- 🔍 支持按名称搜索

### 2. 新增难度
- 📝 难度名称（必填，2-50字符）
- ⭐ 星级选择（1-5星，必选）
- 💰 对应积分（必填，0-10000分）
- 🔘 状态选择（启用/禁用）
- 📄 难度说明（可选，最多200字符）

### 3. 编辑难度
- ✏️ 修改难度名称
- 🌟 调整星级
- 💯 更新积分值
- 🔄 切换状态
- 📝 修改说明

### 4. 删除难度
- 🗑️ 删除确认提示
- ⚠️ 防误删保护

### 5. 分页功能
- 📄 支持多种分页大小（10/20/50/100）
- 🔢 总数统计
- ⬅️➡️ 页码跳转

## 📊 数据结构

```javascript
{
  id: Long,              // 主键ID
  name: String,          // 难度名称
  description: String,   // 难度说明
  starts: Byte,          // 星级（1~5）
  score: Integer,        // 对应积分
  status: Byte,          // 状态（1 启用，0 禁用）
  createTime: DateTime,  // 创建时间
  updateTime: DateTime   // 修改时间
}
```

## 🔌 API 接口

### 1. 获取难度列表
- **接口**: `GET /admin/difficulty/list`
- **参数**: 
  - `page`: 页码
  - `pageSize`: 每页大小
  - `name`: 名称（可选）
- **返回**: PageInfo<Difficulty>

### 2. 新增难度
- **接口**: `POST /admin/difficulty/save`
- **参数**: Difficulty 对象（不含 id）

### 3. 更新难度
- **接口**: `POST /admin/difficulty/update`
- **参数**: Difficulty 对象（含 id）

### 4. 删除难度
- **接口**: `DELETE /admin/difficulty/delete/{id}`
- **参数**: id（路径参数）

## 🚀 使用说明

### 1. 访问页面
在左侧菜单中点击 **内容管理 > 积分难度管理**

### 2. 新增难度
1. 点击「新增难度」按钮
2. 填写必填信息（名称、星级、积分）
3. 选择状态（默认启用）
4. 可选填写说明
5. 点击「确定」保存

### 3. 编辑难度
1. 点击列表中的「编辑」按钮
2. 修改相关信息
3. 点击「确定」保存

### 4. 删除难度
1. 点击列表中的「删除」按钮
2. 确认删除操作
3. 系统自动刷新列表

### 5. 搜索难度
1. 在搜索框输入难度名称
2. 点击「搜索」按钮或按 Enter 键
3. 点击「重置」清空搜索条件

## ⚠️ 注意事项

1. **星级设置**: 星级范围为 1-5，建议按难度递增设置
2. **积分设置**: 积分范围为 0-10000，建议与星级成正比
3. **状态管理**: 禁用的难度等级可能影响相关功能
4. **删除操作**: 删除前请确认该难度等级未被使用
5. **唯一性**: 建议难度名称具有唯一性和辨识度
