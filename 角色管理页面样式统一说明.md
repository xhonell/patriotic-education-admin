# 角色管理页面样式统一说明

## 📋 更新概述

将角色管理页面的样式和布局调整为与用户列表完全统一的风格，提升整体视觉一致性和用户体验。

## ✨ 主要变更

### 1. 页面头部
**之前**：无页面头部，直接显示搜索卡片

**现在**：添加统一的页面头部
```vue
<div class="page-header">
  <h1 class="page-title gradient-text">角色管理</h1>
  <p class="page-subtitle">管理系统角色及权限分配</p>
</div>
```

**样式特点**：
- 大标题：32px，渐变文字效果
- 副标题：14px，灰色说明文字
- 与用户列表、其他管理页面风格完全一致

### 2. 工具栏布局
**之前**：使用 `el-card` 包裹搜索表单

**现在**：使用 `.toolbar.card` 类，与用户列表一致
```vue
<div class="toolbar card">
  <el-form :inline="true" :model="searchForm" class="search-form">
    <!-- 搜索项 -->
    <el-form-item style="margin-left: auto;">
      <el-button type="primary" :icon="Plus" @click="handleAdd">
        新增角色
      </el-button>
    </el-form-item>
  </el-form>
</div>
```

**布局特点**：
- 搜索项在左侧
- 新增按钮通过 `margin-left: auto` 靠右对齐
- 使用 `display: flex` 实现灵活布局
- 响应式自动换行

### 3. 表格容器
**之前**：使用 `el-card` 并带有 header 插槽

**现在**：简化为 `.card` 类，无额外 header
```vue
<div class="card">
  <el-table :data="roleList" v-loading="loading" stripe>
    <!-- 表格列 -->
  </el-table>
</div>
```

**变更原因**：
- 表格标题移至页面头部的副标题
- 新增按钮移至工具栏右侧
- 减少嵌套层级，结构更清晰

### 4. 操作按钮
**之前**：文字链接按钮
```vue
<el-button type="primary" link @click="handleEdit(row)">
  <el-icon><Edit /></el-icon>
  编辑
</el-button>
```

**现在**：圆形图标按钮 + Tooltip
```vue
<div class="action-buttons">
  <el-tooltip content="编辑角色" placement="top">
    <el-button
      type="warning"
      :icon="Edit"
      circle
      size="small"
      @click="handleEdit(row)"
    />
  </el-tooltip>
  <el-tooltip content="删除角色" placement="top">
    <el-button
      type="danger"
      :icon="Delete"
      circle
      size="small"
      @click="handleDelete(row)"
    />
  </el-tooltip>
</div>
```

**样式特点**：
- 圆形按钮（`circle`）
- 渐变背景色
- 悬停上浮效果（`translateY(-2px)`）
- 阴影效果增强交互感
- 使用 Tooltip 显示操作说明

### 5. 按钮渐变样式
```css
/* 警告按钮（编辑） */
.action-buttons .el-button--warning {
  background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
  border: none;
}

.action-buttons .el-button--warning:hover {
  background: linear-gradient(135deg, #f5576c 0%, #f093fb 100%);
}

/* 危险按钮（删除） */
.action-buttons .el-button--danger {
  background: linear-gradient(135deg, #fa709a 0%, #fee140 100%);
  border: none;
}

.action-buttons .el-button--danger:hover {
  background: linear-gradient(135deg, #fee140 0%, #fa709a 100%);
}
```

### 6. 表格样式
**移除**：`border` 属性
**保留**：`stripe` 属性（斑马纹）

**表头渐变**：
```css
:deep(.el-table th) {
  background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
}
```

**行悬停效果**：
```css
:deep(.el-table tr:hover) {
  background-color: rgba(211, 47, 47, 0.05);
}
```

### 7. 搜索按钮样式
```css
/* 主按钮渐变 */
.search-form .el-button--primary {
  background: linear-gradient(135deg, var(--primary-color) 0%, var(--primary-dark) 100%);
  border: none;
}

.search-form .el-button--primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(211, 47, 47, 0.3);
}

/* 次要按钮边框 */
.search-form .el-button:not(.el-button--primary) {
  border: 2px solid var(--border-color);
}

.search-form .el-button:not(.el-button--primary):hover {
  border-color: var(--primary-color);
  color: var(--primary-color);
  transform: translateY(-2px);
}
```

### 8. 对话框样式增强
```css
:deep(.el-dialog) {
  max-width: 800px;
  border-radius: 12px;
}

:deep(.el-dialog__header) {
  padding: 20px 24px;
  border-bottom: 1px solid var(--border-color);
  background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
}

:deep(.el-dialog__title) {
  font-size: 18px;
  font-weight: 600;
  color: var(--text-primary);
}

:deep(.el-dialog__body) {
  padding: 24px;
}
```

**特点**：
- 圆角边框
- 渐变头部背景
- 标题加粗
- 合理的内边距

### 9. 分页样式
```css
:deep(.el-pagination .el-pager li.is-active) {
  background: linear-gradient(135deg, var(--primary-color) 0%, var(--primary-dark) 100%);
}

:deep(.el-pagination button:hover) {
  color: var(--primary-color);
}
```

## 🎨 视觉效果

### 1. 操作按钮对比
**之前**：
- [✏️ 编辑] [🗑️ 删除]（文字链接）

**现在**：
- 🟣 ⚠️ 🔴（圆形图标按钮，带渐变）
  - 鼠标悬停：上浮 + 阴影

### 2. 颜色方案
- **主色调**：红色渐变 (`var(--primary-color)`)
- **编辑按钮**：粉紫渐变 (#f093fb → #f5576c)
- **删除按钮**：粉黄渐变 (#fa709a → #fee140)
- **表头**：灰蓝渐变 (#f5f7fa → #c3cfe2)

### 3. 交互动画
- ✅ 按钮悬停：上浮 2px
- ✅ 按钮悬停：阴影增强
- ✅ 表格行悬停：背景高亮
- ✅ 所有过渡：0.3s ease
