# 政治面貌管理功能说明 🏷️

## 📋 功能概述

在内容管理模块中新增了**政治面貌管理**功能，用于管理系统中用户的政治面貌标签，如"中共党员"、"共青团员"、"群众"等。

## 🎯 核心功能

### 1. **政治面貌列表**
- ✅ 展示所有政治面貌标签
- ✅ 支持按排序号排序
- ✅ 显示创建和更新时间
- ✅ 状态开关控制（启用/禁用）

### 2. **搜索筛选**
- 📝 按名称搜索
- 🔘 按状态筛选（启用/禁用）

### 3. **政治面貌管理**
- ➕ 新增政治面貌
- ✏️ 编辑政治面貌
- 🗑️ 删除政治面貌
- 🔄 状态切换

## 🔧 后端接口

### 接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/admin/politic/list` | GET | 获取政治面貌列表（分页） |
| `/admin/politic/save` | POST | 新增政治面貌 |
| `/admin/politic/update` | POST | 更新政治面貌 |
| `/admin/politic/status/{id}` | POST | 更新状态 |
| `/admin/politic/delete/{id}` | DELETE | 删除政治面貌 |

```javascript
const loadData = async () => {
  const params = {
    page: pagination.page,
    pageSize: pagination.pageSize,
    name: searchForm.name,
    status: searchForm.status
  }
  const res = await getPoliticList(params)
  politicList.value = res.list
  pagination.total = res.total
}
```

#### 2. 新增/编辑
```javascript
const handleSubmit = async () => {
  if (form.id) {
    await updatePolitic(form)
  } else {
    await addPolitic(form)
  }
  loadData()
}
```

#### 3. 状态切换
```javascript
const handleStatusChange = async (row) => {
  await updatePoliticStatus({
    id: row.id,
    status: row.status
  })
}
```

#### 4. 删除
```javascript
const handleDelete = (row) => {
  ElMessageBox.confirm('确定删除？')
    .then(async () => {
      await deletePolitic(row.id)
      loadData()
    })
}
```

## 📝 注意事项

### 名称规范
- ⚠️ 长度：2-50个字符
- ⚠️ 建议使用标准政治面貌名称
- ⚠️ 避免重复名称

### 说明字段
- ⚠️ 可选填写，最多200字符
- ⚠️ 用于描述该政治面貌的详细信息
- ⚠️ 支持多行文本输入

### 状态控制
- ⚠️ 后端使用 Byte 类型（1启用/0禁用）
- ⚠️ 前端自动转换为 boolean（显示开关）
- ⚠️ 禁用不会删除数据，可随时重新启用
- ⚠️ 状态切换即时生效
