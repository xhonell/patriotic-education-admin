## 🚀 快速启动

### 方式一：使用批处理脚本（Windows）

1. 双击 `install.bat` 安装依赖
2. 双击 `start.bat` 启动项目
3. 访问 http://localhost:3000

### 方式二：使用命令行

```bash
# 1. 安装依赖
npm install

# 2. 启动项目
npm run dev

# 3. 访问系统
# 浏览器打开 http://localhost:3000
# 用户名: admin
# 密码: 123456
```
